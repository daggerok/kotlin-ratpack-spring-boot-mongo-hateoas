This repository contains mixed app using ratpack, spring-boot
with mongodb and HATEOAS REST API repository written on kotlin.
And it's f*cking awesome! =)

.run
----
bash gradlew clean build
bash build/libs/*.jar

# ratpack REST API:
http :5050
http :5050/Max
http :5050/skip/x
http :5050/skip/f/b

# mongo hateoas repo:
http :8080
http :8080/users
----
